# 微趣帖内容 SOP（moment）

## 检索策略

### 内容源


| 平台   | 适用场景   | 检索词模板             |
| ---- | ------ | ----------------- |
| 小红书  | 短分享、心得 | `{实体名} 一日游/体验/推荐` |
| 微博   | 即时感受   | `#{实体名}旅行# 推荐`    |
| 今日头条 | 短资讯    | `{实体名} 推荐理由`      |


## 下载规则

1. 每实体目标 10+ 条优质 UGC
2. 存储：`sources/{实体名}/moments/moment_NN.md`

## 质量标准


| 维度  | 阈值         |
| --- | ---------- |
| 字数  | 50-300 字   |
| 原创性 | 真实体验，非广告   |
| 信息量 | 含 1+ 实用信息点 |


## 准出 gate

- 每条 50-300 字
- manifest 含 entityRefs, tagRefs（中文）
- 可含 0-9 张图片引用
