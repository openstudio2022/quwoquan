# 景区主页模板使用指南

## 模板用途

本模板用于生成旅游景区类实体的主页（page.md），包括自然景观、人文景观、主题公园等。

## 适用范围

- 实体类型标签为 `实体类型/地点/景区` 的所有实体
- 包括但不限于：山岳型、湖泊型、古镇型、主题公园型、宗教圣地型

## 生成要求

- 总字数 >= 800 字
- 必须包含 template.md 中定义的所有一级章节
- 每个章节有具体的字数下限
- 所有信息来源于权威百科或实际考证，禁止编造
- 标签和实体引用嵌入正文中，使用 `/tag/` 和 `/entity/` 链接格式
- 图片引用使用 `asset://` 协议
- 禁止出现独立的 `## 标签` 或 `## 相关实体` 章节

## 质量约束

| 维度   | 要求                   |
| ---- | -------------------- |
| 概况   | 含经纬度范围、海拔、面积等可量化信息   |
| 历史沿革 | 至少 3 个关键时间节点         |
| 核心景点 | >= 3 个子景点，每个 >= 80 字 |
| 实用信息 | 门票/开放时间/交通/最佳季节四项必填  |
| 文化特色 | 至少关联 1 个非地理标签        |

## 引用规范

- 实体引用嵌入正文：`[峨眉山](/entity/地点/景区/峨眉山)`
- 标签引用嵌入正文：`[佛教文化](/tag/主题/佛教文化)`
- 图片：`![描述](asset://filename.jpg)` 或 `{asset://filename.jpg|wrapLeft|描述|width=45%}`
