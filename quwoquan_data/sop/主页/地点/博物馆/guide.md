# 博物馆主页模板使用指南

## 模板用途

本模板用于生成博物馆类实体的主页，包括历史博物馆、自然博物馆、科技馆、纪念馆等。

## 适用范围

- 实体类型标签为 `实体类型/地点/博物馆` 的所有实体

## 生成要求

- 总字数 >= 800 字
- 必须涵盖馆藏概况、镇馆之宝、参观动线
- 标签和实体引用嵌入正文
- 禁止出现独立的标签或相关实体章节

## 质量约束

| 维度 | 要求 |
| ---- | ---- |
| 概况 | 含建馆年份、面积、馆藏数量 |
| 镇馆之宝 | >= 3 件，每件含年代/材质/特色 |
| 参观动线 | 推荐路线含时长 |
| 实用信息 | 开放时间/门票/预约方式 |
