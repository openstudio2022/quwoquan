# 住宿主页模板使用指南

## 模板用途

本模板是住宿类实体主页（page.md）的通用基础模板，适用于所有住宿业态。针对酒店、民宿、度假村另有专项模板。

## 适用范围

- 实体 tagRefs 中包含 `Entity/地点/住宿/住宿业态/*` 的所有实体
- 未被 `酒店`、`民宿`、`度假村` 专项模板覆盖的住宿类型（如青年旅舍、客栈、农家乐、营地住宿等）

## 生成要求

- 总字数 >= 600 字
- 必须包含 template.md 中定义的所有一级章节
- 信息来源必须有至少一个可信锚点（OTA页面、官网、地图POI、百科）
- 标签和实体引用嵌入正文中，使用 `/tag/` 和 `/entity/` 链接格式
- 图片引用使用 `asset://` 协议

## 质量约束

| 维度     | 要求                                  |
| ------ | ----------------------------------- |
| 基本信息   | 含地址、开业年份、房间数/床位数等可量化信息              |
| 业态标签   | 至少引用一个 Entity/地点/住宿/住宿业态/* 叶子标签     |
| 档次标签   | 若可确认，引用一个 Entity/地点/住宿/档次等级/* 叶子标签  |
| 功能标签   | 若有明确定位，引用一个或多个 Entity/地点/住宿/功能定位/*  |
| 设施服务   | 至少列出 3 项核心设施或服务                     |
| 行政区标签  | 必须含行政区标签如 Topic/地理/行政区/中国/四川省/成都市   |
| 来源可信度  | 必须有至少一个来源锚点（OTA/官网/地图/百科），无锚点不得发布  |

## 引用规范

- 实体引用：`[实体名](/entity/地点/{业态}/{名称})`
- 标签引用：`[酒店](/tag/Entity/地点/住宿/住宿业态/酒店)`
- 行政区引用：`[成都市](/tag/Topic/地理/行政区/中国/四川省/成都市)`
- 图片：`![描述](asset://filename.jpg)` 或 `{asset://filename.jpg|wrapLeft|描述|width=45%}`
