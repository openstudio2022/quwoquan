# 民宿主页模板使用指南

## 模板用途

本模板用于生成民宿类实体的主页（page.md），突出民宿的在地体验与主人文化特色。

## 适用范围

- 实体 tagRefs 中包含 `Entity/地点/住宿/住宿业态/民宿` 或 `Entity/地点/住宿/住宿业态/客栈` 的实体
- 包括但不限于：乡村民宿、城市民宿、古镇客栈、藏式民居、庭院客栈等

## 生成要求

- 总字数 >= 600 字
- 必须包含 template.md 中定义的所有一级章节
- 信息来源必须有至少一个可信锚点（Airbnb/美团民宿/途家/小红书/地图）
- 重点突出"在地体验"与"主人文化"，与标准酒店模板区分

## 质量约束

| 维度     | 要求                                    |
| ------ | ------------------------------------- |
| 基本信息   | 含地址、房源形态、房间数、开业/运营年份                 |
| 业态标签   | 必须含 Entity/地点/住宿/住宿业态/民宿 或 客栈        |
| 房源形态   | 至少引用一个 Entity/地点/住宿/房源形态/* 叶子          |
| 在地体验   | 描述与当地文化/自然的关联，不少于 150 字               |
| 主人故事   | 简述民宿主人背景与经营理念（可选但推荐）                 |
| 设施服务   | 至少列出 3 项核心设施                          |
| 行政区标签  | 必须含精确到县/区的行政区标签                       |
| 来源可信度  | 至少一个来源锚点                              |

## 引用规范

- 实体引用：`[实体名](/entity/地点/民宿/{名称})`
- 标签引用：`[整套房源](/tag/Entity/地点/住宿/房源形态/整套房源)`
- 图片：`![描述](asset://filename.jpg)`
