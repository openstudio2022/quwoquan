# 客栈主页模板使用指南

## 模板用途

本模板用于生成传统院落/胡同（巷子）语境下的小型客栈类实体主页，突出在地文化与居住氛围。

## 适用范围

- 实体类型路径为 `/entity/地点/客栈/{{实体名}}` 的条目
- 多见于古城、街巷、四合院/天井院落改造，房型规模通常小于综合体酒店

## 生成要求

- 总字数 >= 800 字
- 明确「地域文化叙事」：建筑形制、纹样/材质、与当地生活方式的耦合点不少于 3 条
- 房型与公共空间（院落、天井、露台、书房茶席）需写清规模与体感，避免泛泛「温馨舒适」

## 标签与引用

- 正文内嵌 **[tag](/tag/Topic/...)** 与 **[实体](/entity/地点/客栈/...) 等跳转**，不出现单独罗列的「标签章节」「相关实体附录」块
- 关联景点/街巷使用对应实体路径（如古镇、步行街），禁止再用旧路径 `Entity/地点/住宿/住宿业态/*`
