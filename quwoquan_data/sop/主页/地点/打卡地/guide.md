# 打卡地主页模板使用指南

## 模板用途

本模板用于生成网络热门或小众特色目的地的主页，如网红地标、隐藏街巷、特色街区等。

## 适用范围

- 实体类型标签为 `实体类型/地点/打卡地` 的所有实体
- 包括但不限于：商业综合体、文创街区、网红墙/建筑、小众秘境

## 生成要求

- 总字数 >= 800 字
- 所有章节含量化信息（面积、年客流、开业时间等）
- 标签和实体引用嵌入正文，使用 `/tag/` 和 `/entity/` 链接格式
- 图片引用使用 `asset://` 协议
- 禁止出现独立的标签或相关实体章节

## 质量约束

| 维度   | 要求 |
| ---- | ---- |
| 概况   | 含地址、面积/规模、开放年份 |
| 为什么火 | 至少2个传播因素 |
| 打卡攻略 | >= 3 个具体拍照/体验点 |
| 实用信息 | 营业时间/交通/费用/最佳时段 |
| 周边联动 | 至少1个关联实体 |
