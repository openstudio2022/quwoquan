# 酒店主页模板使用指南

## 模板用途

本模板用于生成酒店类实体的主页（page.md），专注于标准商业酒店的全面展示。

## 适用范围

- 实体 tagRefs 中包含 `Entity/地点/住宿/住宿业态/酒店` 的实体
- 包括但不限于：商务酒店、会议酒店、精品酒店、连锁酒店等

## 生成要求

- 总字数 >= 800 字
- 必须包含 template.md 中定义的所有一级章节
- 信息来源必须有至少一个可信锚点（携程/美团/官网/地图POI）
- 标签和实体引用嵌入正文中

## 质量约束

| 维度     | 要求                                    |
| ------ | ------------------------------------- |
| 基本信息   | 含品牌、星级/档次、地址、开业年份、翻新年份、房间总数          |
| 业态标签   | 必须含 Entity/地点/住宿/住宿业态/酒店              |
| 档次标签   | 必须含一个 Entity/地点/住宿/档次等级/* 叶子           |
| 功能定位   | 至少一个 Entity/地点/住宿/功能定位/* 叶子            |
| 设施服务   | 至少列出 5 项核心设施，引用对应标签                   |
| 房型     | 至少介绍 3 种房型，引用对应房型标签                   |
| 行政区标签  | 必须含精确到市/区的行政区标签                       |
| 商务配套   | 若为商务酒店，必须描述会议室、行政酒廊等                 |
| 来源可信度  | 至少一个来源锚点                              |

## 引用规范

- 实体引用：`[实体名](/entity/地点/酒店/{名称})`
- 标签引用：`[五星级](/tag/Entity/地点/住宿/档次等级/五星级)`
- 图片：`![描述](asset://filename.jpg)`
