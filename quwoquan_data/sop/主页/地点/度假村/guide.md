# 度假村主页模板使用指南

## 模板用途

本模板用于生成度假村类实体的主页（page.md），突出综合度假体验与休闲活动。

## 适用范围

- 实体 tagRefs 中包含 `Entity/地点/住宿/住宿业态/度假村` 的实体
- 包括但不限于：温泉度假村、滑雪度假村、高尔夫度假村、一价全包度假村等

## 生成要求

- 总字数 >= 800 字
- 必须包含 template.md 中定义的所有一级章节
- 信息来源必须有至少一个可信锚点（官网/OTA/旅游媒体报道）
- 重点突出"综合度假体验"与"活动丰富度"

## 质量约束

| 维度     | 要求                                    |
| ------ | ------------------------------------- |
| 基本信息   | 含品牌、占地面积、客房总数、开业年份                   |
| 业态标签   | 必须含 Entity/地点/住宿/住宿业态/度假村             |
| 功能定位   | 至少一个 Entity/地点/住宿/功能定位/* 叶子（如度假住宿）    |
| 度假活动   | 至少介绍 4 种可体验活动                         |
| 设施服务   | 至少列出 5 项核心设施                          |
| 房型     | 至少介绍 3 种房型/别墅类型                       |
| 自然环境   | 描述度假村所处自然环境与景观                        |
| 行政区标签  | 必须含精确到县/区的行政区标签                       |
| 来源可信度  | 至少一个来源锚点                              |

## 引用规范

- 实体引用：`[实体名](/entity/地点/度假村/{名称})`
- 标签引用：`[度假住宿](/tag/Entity/地点/住宿/功能定位/度假住宿)`
- 图片：`![描述](asset://filename.jpg)`
