# 遗址主页模板使用指南

## 模板用途

本模板用于生成遗址类实体的主页，包括古代遗存、工业遗产、近现代遗址等。

## 适用范围

- 实体类型标签为 `实体类型/地点/遗址` 的所有实体

## 生成要求

- 总字数 >= 800 字
- 必须涵盖历史沿革、核心看点、文化价值
- 标签和实体引用嵌入正文
- 禁止出现独立的标签或相关实体章节

## 质量约束

| 维度 | 要求 |
| ---- | ---- |
| 概况 | 含始建年代、面积、保护等级 |
| 历史沿革 | >= 3 个关键节点 |
| 核心看点 | >= 3 处遗存或特色 |
| 文化价值 | 至少关联 1 个非地理主题标签 |

## 引用规范

- 实体引用嵌入正文：`[东风堰](/entity/地点/遗址/东风堰)`
- 标签引用嵌入正文：`[水利工程](/tag/主题/水利工程)`
- 图片：`{asset://filename.jpg|wrapLeft|描述|width=45%}`
