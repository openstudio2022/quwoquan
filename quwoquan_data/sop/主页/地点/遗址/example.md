# 东风堰

> 始建于清康熙年间的世界灌溉工程遗产，四川盆地边缘山丘区引水灌溉的典范

东风堰全貌

## 概况

东风堰位于四川省乐山市夹江县境内，青衣江左岸。干渠全长16.2公里，灌溉面积7万余亩，覆盖夹江县城及周边农田。2014年列入首批世界灌溉工程遗产名录。渠首位于千佛岩大弯处，利用天然弯道的水位差自流引水，无需机械提水，体现了古人因地制宜的水利智慧。

## 历史断代

- **清康熙元年（1662年）**：始建，最初名为"毛滩堰"，由当地士绅集资开凿，初期仅灌田300余亩。
- **清乾隆、嘉庆年间**：多次扩建加固，灌溉面积扩至万余亩，更名为"东风堰"。渠系从单渠发展为干支相连的灌溉网络。
- **1950年代**：新中国成立后纳入国家水利管理体系，进行系统整治，干渠全线衬砌，灌溉面积扩至7万亩。
- **2014年9月**：在韩国光州召开的国际灌排委员会第65届执行理事会上，列入首批世界灌溉工程遗产名录（全球仅17处）。

## 营造特征

{asset://东风堰_detail.jpg|wrapRight|渠首引水口|width=40%}

东风堰渠首设置于青衣江千佛岩河段大弯道凸岸侧，利用弯道横向环流原理，使含沙量低的表层水自流入渠，泥沙随底层水流继续沿主河道排走——这是一种无坝引水方式，与都江堰的"鱼嘴分水"异曲同工。干渠沿山丘等高线蜿蜒，平均纵坡约1/3000，依靠地形自然落差实现全线自流灌溉。渠道断面以梯形为主，底宽2.5-4米，水深1.2-1.8米。沿线设有分水闸、泄洪闸共12座，退水渠4条。

| 保护级别   | 世界灌溉工程遗产 / 四川省文物保护单位                          |
| 列入时间   | 2014年（世界灌溉工程遗产首批）                             |
| 主要研究成果 | ①无坝引水技术研究——弯道取水的水力学分析；②清代川西水利社会组织研究——以东风堰堰规为例 |
| 现状     | 保存完好，至今仍在发挥灌溉功能，同时开辟为水利遗产旅游线路                 |
