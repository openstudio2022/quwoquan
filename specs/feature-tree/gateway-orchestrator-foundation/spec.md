# L1 特性：gateway-orchestrator-foundation

## 功能说明
- 提供网关统一入口、鉴权限流、防护策略与跨服务编排基础能力。
- 提供上下文透传、降级回滚和链路追踪能力，保障端云链路稳定。

## 约束
- 网关为端侧唯一入口，业务服务不得直连暴露。
- 编排输出结构必须稳定并支持兼容窗口。
- request/trace/page/session/user/device 字段必须全链路透传。

## 验收标准
- A3：鉴权、限流、降级、回滚策略生效。
- A4：网关与编排链路可观测可告警。
- A7：接口与 headers 契约一致。
- A8：冒烟/集成/UAT 自动化验证通过。
