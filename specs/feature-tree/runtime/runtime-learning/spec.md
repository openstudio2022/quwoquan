# L2 特性：runtime-learning

## 功能说明
- 提供统一反馈事件、评分卡、评估记录与优化闭环版本化模型。
- 支撑推荐与助手等能力的自动化优化（采集->评估->发布->回滚）。

## 约束
- 事件字段与版本元数据必须对齐 `contracts/feedback_and_learning.md`。
- 反馈采集必须可追踪可审计，不得丢失关键关联字段。
- 优化闭环必须支持灰度与回滚，不能直接全量生效。

## 验收标准
- A1：反馈事件模型与写入接口可复用。
- A5：具备可执行的优化闭环链路。
- A7：字段契约与模型一致。
- A8：事件/评估/版本化自动化测试完整。
