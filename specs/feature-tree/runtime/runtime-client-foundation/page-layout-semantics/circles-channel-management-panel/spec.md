# L4 契约：circles-channel-management-panel

## 功能说明

圈子页将右上角助理头像替换为频道管理三横图标；点击后在一级 tab 下方滑出频道管理面板，交互与布局遵循微博频道编辑样式。频道源以当前 app 圈子一级 tab 为准，并支持本地持久化。

## 范围

- 入口：`CirclesPage` 顶部 `trailingActions` 改为频道管理图标（非助理头像）
- 面板层级：紧贴一级 tab 下方滑出（非全屏路由）
- 对齐要求：圈子一级 tab 必须左对齐并与内容区左边缘一致（使用内容区同一水平内边距语义）
- 模块区块：
  - 我的频道（已选）：支持拖拽排序、点击 `x` 移到未选
  - 全部频道（未选）：点击 `+` 追加到已选末尾
- 默认策略：`car`、`humanity`、`sports` 首装默认在未选
- 持久化：本地保存频道顺序与选中状态，重启后恢复
- 视觉约束：主操作与强调色使用蓝色主题（`AppColors.primaryColor`），禁止橘色动作色
- 稳定性约束：一级 tab 在滚动、切换、面板展开/收起过程中不得出现位置跳变

## 与父节点关系

父节点：`page-layout-semantics` L3

## 验收标准

- 入口图标为三横频道管理图标；点击后面板在一级 tab 下方展开/收起
- 已选/未选频道在操作后即时互转，且 `+` 进入已选末尾、`x` 进入未选
- 已选频道支持拖拽调整顺序，一级 tab 顺序同步变化
- `car`、`humanity`、`sports` 首装默认不在「我的频道」
- 重启后频道顺序与选中状态保持一致
- 面板中“完成”等主动作呈现为蓝色主题
- 一级 tab 左边缘与内容区对齐，滚动与切换过程中不出现突跳（含锚定切换边界）
