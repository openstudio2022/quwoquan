# L1 规格：圈子社区

## 范围
- 圈子 CRUD、活动流、成员权限、圈内推荐与反馈闭环。

## 功能说明
- 提供圈子生命周期能力（创建、查询、运营、治理），支撑社区组织与内容分发。
- 提供圈子内活动流和推荐能力，结合行为反馈做持续优化。
- 建立成员与权限模型，保证圈内协作、管理与安全边界可控。

## 约束
- 圈子活动流接口必须支持游标分页与推荐排序。
- 成员与权限变更必须可审计，且错误码可定位模块与原因。
- 圈子行为事件必须进入运营事件体系，支撑推荐优化。

## 验收标准（L1 重点）
- A1：圈子列表/详情/成员管理/活动流完整可用。
- A3：权限校验、限流、降级策略可配置。
- A5：圈内事件采集与优化闭环可运行。
- A8：自动化测试覆盖核心流程。
