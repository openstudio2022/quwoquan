# L3 规格：chat-group-admin-govern — 群聊管理与权限治理

> **层级**：L3_story（隶属 L2 `chat-experience-optimization`）
> **状态**：specified

## 0. 一句话定义

群聊管理页（群主/管理员专属）、群主转让、管理员设置（最多 3 人）、群名称修改权限控制、隐私屏障权限控制，实现群聊管理的分级治理闭环。

## 1. 背景与动机

当前群聊设置页（`ChatSettingsPage`）存在以下缺失：

1. **无群管理页**：群主和管理员缺少专属管理入口，无法设置进群策略、转让群主、管理管理员。
2. **群名称无权限控制**：任何成员都能修改群名称，缺少"仅管理员可修改"开关。
3. **隐私屏障无权限控制**：任何成员都能开关隐私屏障，应由管理员控制。
4. **无群主转让**：群主无法将管理权限转让给其他成员。
5. **无管理员设置**：无法指定/取消管理员。

## 2. 目标用户

- 群主：拥有最高管理权限
- 管理员（最多 3 人）：群主指定的二级管理者
- 普通成员：受管理员设置约束

## 3. 功能范围

### 3.1 In-Scope

| 编号 | 功能 | 说明 |
|---|---|---|
| GA1 | 群管理页 | 群主/管理员可见，含以下设置项 |
| GA2 | 二维码进群 | 开关，控制是否允许扫码进群 |
| GA3 | 进群需确认 | 开关，进群是否需要群主/管理员审批 |
| GA4 | 仅管理员改群名 | 开关，控制群名称修改权限 |
| GA5 | 群主管理权转让 | 群主选择一位成员转让管理权，转让后自动变为普通成员 |
| GA6 | 管理员设置 | 群主设置/取消管理员，最多 3 人 |
| GA7 | 解散群聊 | 仅群主可操作的危险操作 |
| GA8 | 群名称权限拦截 | 非管理员点击群名称时弹窗提示"群组已设定为只有群主或管理员才能修改群名" |
| GA9 | 隐私屏障权限 | 仅管理员可开关，非管理员为 disabled 不可修改状态 |
| GA10 | 设置页头像方形化 | 群设置页成员头像改为圆角方形，尺寸比对话页大一号（52px） |

### 3.2 Out-of-Scope

- 群公告编辑（后续独立特性）
- 群文件管理
- 成员禁言（后续特性）
- 群举报/拉黑（已由 `group-settings` 冻结为下沉到对象层）

## 5. 权限模型

| 操作 | 群主 | 管理员 | 普通成员 |
|---|---|---|---|
| 进入群管理页 | 可以 | 可以 | 不可见 |
| 二维码进群开关 | 可以 | 可以 | — |
| 进群需确认开关 | 可以 | 可以 | — |
| 仅管理员改群名 | 可以 | 可以 | — |
| 转让群主 | 可以 | 不可见 | — |
| 设置管理员 | 可以 | 不可见 | — |
| 解散群聊 | 可以 | 不可见 | — |
| 修改群名称 | 始终可以 | 始终可以 | 取决于开关 |
| 隐私屏障开关 | 可以 | 可以 | disabled（灰色不可操作） |

## 6. 端云一致性

### 6.1 新增字段（ConversationGroupSettings）

```yaml
groupSettings:
  qrCodeJoinEnabled: boolean      # 是否允许扫码进群
  joinRequiresApproval: boolean   # 进群是否需审批
  nameEditableByAdminOnly: boolean # 仅管理员可改群名
  privacyShieldAdminOnly: boolean  # 隐私屏障仅管理员可控
```

### 6.2 新增/扩展 API

| 接口 | 方法 | 说明 |
|---|---|---|
| `PATCH /conversations/{id}/settings` | PATCH | 更新群设置（含上述 4 个开关） |
| `GET /conversations/{id}/settings` | GET | 获取群设置 |
| `PATCH /conversations/{id}/owner` | PATCH | 转让群主（body: `{newOwnerId}`) |
| `PUT /conversations/{id}/admins` | PUT | 设置管理员列表（body: `{adminIds: [...]}`) |
| `DELETE /conversations/{id}` | DELETE | 解散群聊（仅群主） |

### 6.3 WebSocket 事件

| 事件 | 触发场景 | 说明 |
|---|---|---|
| `conv.settings.updated` | 群设置变更 | 推送最新设置到所有成员 |
| `conv.owner.transferred` | 群主转让 | 推送新旧群主信息 |
| `conv.admins.updated` | 管理员变更 | 推送最新管理员列表 |
| `conv.dissolved` | 群被解散 | 通知所有成员 |

### 6.4 变更通知

群名称修改成功后，通知所有群成员并同步：
- WebSocket 推送 `conv.updated` 事件
- 其他成员的会话缓存自动更新

## 7. 对标

| 维度 | 微信 | 趣聊目标 | 借鉴/不借鉴 |
|---|---|---|---|
| 群管理页 | 独立页面，群主可见 | 独立页面，群主+管理员可见 | 借鉴 |
| 管理员数量 | 最多 3 人 | 最多 3 人 | 完全借鉴 |
| 群主转让 | 选择成员 + 确认弹窗 | 选择成员 + 确认弹窗 | 完全借鉴 |
| 群名称权限 | 管理员可控 | 管理员可控 | 完全借鉴 |

## 8. 业务约束

- 管理员最多 3 人（不含群主）
- 群主转让后原群主自动变为普通成员
- 解散群聊为不可逆操作
- 群名称修改权限受 `nameEditableByAdminOnly` 开关控制
- 隐私屏障受 `privacyShieldAdminOnly` 开关控制，非管理员看到 disabled 状态
- 确认弹窗必须屏幕上下左右居中
- 主题色蓝色

## 9. 非功能要求

- 群设置变更后 ≤2s 推送到所有在线成员
- 群主转让为原子操作，不存在"两个群主"的中间态
