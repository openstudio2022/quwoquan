# L3 特性：moment-display-journey（微趣旅程）

## 功能说明

**微趣独立旅程**：微趣频道 → 微趣详情/浏览器 → 作者详情，端到端数据源一致、交互状态跨页面同步、重入状态保持。按 category=moment 隔离。

- **数据源**：列表、微趣详情/浏览器、作者详情共用同一 feed
- **展示**：微趣详情与 post 一致（作者、关注、赞、收藏、评论数、转发、内容）
- **状态**：关注/赞/收藏跨页面同步；作者详情关注变更返回后同步；重入状态保持
- **滑动**：若微趣有浏览器形态，滑动顺序与列表一致；支持加载更多

## 适用范围与约束

- **适用**：发现页微趣频道；微趣详情/浏览器；作者详情
- **约束**：按 category=moment 隔离；微趣浏览器/详情形态待确定
- **当前**：可暂缓实现，待图片、视频、文章完成后扩展

## 验收

- A1：列表、微趣详情、作者详情数据与 post 一致
- A2：关注/赞/收藏跨页面一致，重入后保持
- A3：作者详情关注变更返回同步
- A4：若有滑动，顺序与列表一致
- A6：Mock/Remote 切换后正确
