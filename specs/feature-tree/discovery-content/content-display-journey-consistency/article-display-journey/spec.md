# L3 特性：article-display-journey（文章旅程）

## 功能说明

**文章独立旅程**：文章频道 → 文章详情页 → 作者详情，端到端数据源一致、交互状态跨页面同步、重入状态保持。按 category=article 隔离。

- **数据源**：列表、文章详情页、作者详情共用同一 feed
- **展示**：文章详情与 post 一致（作者、关注、赞、收藏、评论数、转发、正文）
- **状态**：关注/赞/收藏跨页面同步；作者详情关注变更返回后同步；重入状态保持

## 适用范围与约束

- **适用**：发现页文章频道；文章详情页；作者详情
- **约束**：按 category=article 隔离；文章详情页形态与图片/视频浏览器不同，无左右滑动
- **当前**：可暂缓实现，待图片/视频完成后扩展

## 验收

- A1：列表、文章详情、作者详情数据与 post 一致
- A2：关注/赞/收藏跨页面一致，重入后保持
- A3：作者详情关注变更返回同步
- A6：Mock/Remote 切换后正确
