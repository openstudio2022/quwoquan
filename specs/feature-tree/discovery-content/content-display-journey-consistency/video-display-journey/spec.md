# L3 特性：video-display-journey（视频旅程）

## 功能说明

**视频独立旅程**：视频频道 → 视频沉浸式浏览器 → 作者详情，端到端数据源一致、交互状态跨页面同步、重入状态保持。与图片旅程模式一致，按 category=video 隔离。

- **数据源**：列表、浏览器、作者详情共用同一 feed（category=video）
- **展示**：浏览器展示与 post 一致（作者、关注、赞、收藏、评论数、转发、视频、文字）
- **状态**：关注/赞/收藏跨页面同步；作者详情关注变更返回后同步；退出再进状态保持
- **滑动**：左右滑动顺序与视频频道 feed 一致；滑动到底加载更多

## 适用范围与约束

- **适用**：发现页视频频道；视频沉浸式浏览器（ImmersiveVideoViewer）；作者详情
- **约束**：须支持 Mock/Remote 一键切换；按 category=video 隔离，不与图片混用

## 验收

- A1：列表、浏览器、作者详情数据与 post 一致
- A2：关注/赞/收藏跨页面一致，重入后保持
- A3：作者详情关注/取消后返回，浏览器展示更新
- A4：滑动顺序与列表一致，支持加载更多
- A5：视频旅程能力与图片一致
- A6：Mock/Remote 切换后正确
