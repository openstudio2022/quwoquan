# profile-author

## 新增需求

### 需求：作者主页与 PROFILE_PAGE_DESIGN v3.0 一致

作者（用户）主页叠加层须与设计文档及 AVATAR_DESIGN_SYSTEM 一致：**圆形头像**（avatar-user-xl，个人头像语义）、统计为**关注、粉丝、获赞、圈子**（四栏），主 Tab 为**创作、互动、生活**。创作 Tab 子分类为**全部、图片、视频、文章**（弃用瞬间、随记）；生活 Tab 子分类为**足迹、书影音、味蕾、爱物**（弃用好物橱窗）。使用胶囊样式二级 Tab，支持响应式横滚。

#### 场景：作者布局

- **当** 作者主页被展示时
- **则** 统计展示关注/粉丝/获赞/圈子；主 Tab 为创作/互动/生活；创作子分类为全部/图片/视频/文章；生活子分类为足迹/书影音/味蕾/爱物

#### 场景：返回关闭叠加层

- **当** 用户点击返回
- **则** 作者主页关闭，并恢复前一上下文

### 需求：吸顶与内容区

滚动时主 Tab 栏须吸顶。创作 Tab 下展示作品网格或列表；互动 Tab 下展示赞/评论/互访；生活 Tab 下展示足迹/书影音/味蕾/爱物。支持网格/列表切换。

#### 场景：Tab 吸顶与内容

- **当** 用户滚动作者主页
- **则** Tab 栏吸顶；切换 Tab 时内容区切换为对应内容

### 需求：作者主页使用语义 token

作者主页上所有 UI 须使用 AppColors、AppSpacing、UITextConstants。禁止使用硬编码值。

#### 场景：作者主页的 token

- **当** 作者主页被构建时
- **则** 所有样式与文案使用设计系统常量
