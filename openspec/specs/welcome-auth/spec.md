# welcome-auth

## 新增需求

### 需求：首次进入时展示欢迎页

应用须在用户打开应用且尚未进入主框架时展示欢迎页。欢迎页须与 Figma 的视觉与动效（如淡入/滑动）一致。

#### 场景：用户首次打开应用

- **当** 应用启动且当前会话未标记为「已通过欢迎」
- **则** 全屏展示欢迎页，视觉与 Figma 一致

#### 场景：用户完成欢迎

- **当** 用户在欢迎页执行完成操作（如点击按钮或滑动）
- **则** 欢迎页按既定动效关闭，并展示主应用框架（五大频道）

### 需求：欢迎页仅使用语义 token

欢迎页上所有颜色、间距与文案须使用 AppColors、AppSpacing、UITextConstants（或扩展语义）。禁止使用硬编码值。

#### 场景：主题与 token

- **当** 欢迎页被构建时
- **则** 所有颜色、尺寸与文案均来自设计系统常量
