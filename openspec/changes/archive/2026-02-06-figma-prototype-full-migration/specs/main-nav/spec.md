# main-nav

## 新增需求

### 需求：底部导航展示五个频道

应用须提供底部导航栏，包含五个入口：发现、圈子、创作、趣聊、我的。文案与图标须与 Figma 一致。

#### 场景：五个入口可见

- **当** 用户处于主应用框架内（非叠加层）
- **则** 底部栏展示全部五个入口及正确文案与图标

#### 场景：当前频道高亮

- **当** 用户处于某一频道时
- **则** 对应导航项在视觉上高亮（如激活色/字重），点击其他项可切换频道

### 需求：创作入口打开创作入口抽屉

「创作」入口须打开创作入口抽屉，而不是跳转到页面。点击创作不得改变当前选中的频道索引。

#### 场景：创作打开抽屉

- **当** 用户点击「创作」入口
- **则** 展示创作入口抽屉（如底部抽屉），且底部导航当前选中项保持不变

### 需求：导航使用语义 token

底部导航的所有颜色、间距、图标与文案须使用设计系统（AppColors、AppSpacing、UITextConstants）。禁止使用硬编码值。

#### 场景：导航栏 token

- **当** 底部导航被构建时
- **则** 样式与文案仅来自语义常量
