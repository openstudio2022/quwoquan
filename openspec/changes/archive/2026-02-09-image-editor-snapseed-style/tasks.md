- [x] 1.1 实现图片编辑器页面外壳：顶栏固定、中部占满剩余高度、底栏固定，三区边界清晰无重叠
- [x] 1.2 顶栏左侧返回、中间图片序号、右侧完成按钮（主色蓝色，禁止红色，使用 AppColors）
- [x] 1.3 底栏主工具栏：裁剪、旋转、滤镜、美颜、文字、马赛克、相框、专业修图等平级入口，支持横向滚动
- [x] 1.4 全部 UI 使用 AppColors、AppSpacing、UITextConstants，无硬编码
- [x] 1.5 编辑器整体背景采用深色主题（使用 AppColors.dark.backgroundPrimary/Secondary）

## 2. 顶栏与编辑记录（Snapseed 式）

- [x] 2.1 实现编辑步骤栈：每步记录工具类型与参数快照，支持追加、删除某步、重算其后步骤
- [x] 2.2 顶栏或底栏提供记录入口，打开步骤列表可查看、删除某步、或重做某步（进入该步编辑并保存后更新）
- [x] 2.3 完成按钮点击前可访问记录；点击完成后提交结果并关闭编辑器（与 create-flow 约定一致）

## 3. 中部图片与多图滑动

- [x] 3.1 中部展示当前图片，占满可用垂直空间；单图时顶栏序号为 1/1
- [x] 3.2 多图时中部使用 PageView（或等价）左右滑动切换图片，顶栏序号随当前页更新为「当前/总数」

## 4. 工具面板统一弹出

- [x] 4.1 点击任一底栏工具入口均弹出新面板替代底部区域
- [x] 4.2 面板结构：顶部为分类/选项、中间为模板或内容、底部为取消（叉）与保存（勾）
- [x] 4.3 取消时放弃当前工具子操作并关闭面板；保存时将当前操作作为一步写入记录并关闭面板

## 5. 专业修图面板与联动

- [x] 5.1 进入专业修图面板后展示二级 Tab（曝光 / 色彩 / 光影 / 质感）与横向工具列表
- [x] 5.2 工具列表为连续横向流，无分组标题或分割线
- [x] 5.3 Tab 与工具列表双向联动：滑动列表高亮对应 Tab，点击 Tab 滚动到对应分组起始
- [x] 5.4 点击专业工具进入对应工具面板；叉/勾返回专业面板；再次叉/勾返回主工具栏

## 6. 常规工具面板

- [x] 6.1 选中滤镜：面板顶部滤镜分类、中间滤镜模板列表与预览、底部取消/保存及强度滑块
