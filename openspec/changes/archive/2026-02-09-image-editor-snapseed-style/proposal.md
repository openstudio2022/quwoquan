## Why

当前图片编辑器实现与产品目标差距较大，需要以 Snapseed、醒图等专业图片编辑软件为能力基线重新实现。用户需在完成前拥有完整操作历史（可查看、删除单步、重做单步），顶栏完成按钮为主色蓝色；多图时中部可滑动浏览；旋转/滤镜/曲线等工具需具备专业级交互（旋转仪表与网格、滤镜分类与强度、曲线蒙皮等）。

## What Changes

- **布局**：严格三段式——顶部工具栏（返回、图片序号、完成按钮蓝色）、中部图片展示区（支持多图时左右滑动）、底部编辑功能栏。
- **顶栏**：返回、当前图片序号（如 1/1、2/5，随内容区滑动联动）、完成按钮（AppColors.primaryColor 蓝色，禁止红色）；完成前支持 Snapseed 式历史：记录全部编辑步骤，可查看步骤列表、删除某步或重做某步。
- **中部**：仅展示当前图片；多图时支持左右滑动切换，顶栏序号随之更新。
- **底栏工具（平级入口）**：裁剪、旋转、滤镜、美颜、文字、马赛克、相框等常规工具与「专业修图」入口并列展示；可横向滚动。
- **专业修图**：进入后展示二级分组 Tab（曝光 / 色彩 / 光影 / 质感）与横向专业工具列表（无分组标题或分割线）；Tab 与工具列表双向联动。专业工具包含：局部、亮度、曲线、曝光、对比度、饱和度、HSL、高光、阴影、色调、白平衡、降噪、锐化/去锐化等。
- **操作面板**（底栏选中任一功能后出现）：
  - **通用结构**：统一弹出新面板替代底部区域；底部为取消（叉）与保存（勾），中部为操作模板或内容，顶部为可选分类或选项。
  - **专业修图面板**：选择「专业修图」先进入专业工具面板；点击具体专业工具后进入对应工具面板，点击叉/勾返回专业面板，再次叉/勾返回主工具栏。
  - **滤镜**：顶部滤镜分类、中间滤镜模板列表、底部取消/保存及强度滑块。
  - **旋转**：中间为旋转相关选项，图片下方出现旋转仪表（便于约 ±45° 精细调节）；调节时图片上叠加网格，便于对齐主体与角度。
  - **曲线**：曲线调节相关控件以蒙皮形式叠加在图片下方区域，便于直接调节颜色与明暗。
- **背景与语义设计**：编辑器整体背景采用深色（参考 Snapseed/醒图的黑底编辑器），使用设计系统深色背景语义色；全部使用 AppColors、AppSpacing、UITextConstants，完成按钮为主色蓝，禁止硬编码与魔鬼数字。

## Capabilities

### New Capabilities

（无；本变更在既有 image-editor 能力上做需求补充与细化。）

### Modified Capabilities

- `image-editor`: 在既有三段式布局、历史操作、底栏工具集与操作面板结构基础上，补充与细化：顶栏序号与中部多图滑动联动；完成按钮必须为主色蓝且禁止红色；编辑器整体背景为深色；旋转工具增加「图片下方旋转仪表」与「调节时图片上网格」；专业修图明确包含 HSL 调色；曲线工具以「图片下方蒙皮」形式展示曲线调节界面。

## Impact

- 替换或重构现有图片编辑相关页面与组件（如 CreatePage 内图片编辑入口所打开的编辑器）。
- 实现或对接编辑管线：步骤栈（历史）、多图状态、旋转/裁剪/滤镜/曲线/HSL/局部等渲染与导出。
- 依赖设计系统（AppColors、AppSpacing、UITextConstants）及 create-flow 对图片 Tab 使用本编辑器的约定。
