# 安全、生命周期与清理

## 1. 安全原则

- 对象存储默认私有写
- 客户端只拿有限时 presign
- CDN 域名统一受控
- 敏感资源支持签名 URL

## 2. 上传安全

- 校验 `contentType`
- 校验 `fileSize`
- 禁止用户原始文件名直接入库
- 会话超时自动失效

## 3. 访问安全

- 公开资源使用 CDN 域名
- 私有资源使用签名 URL
- 支持防盗链与 Referer 规则

## 4. 生命周期规则

建议按资产类型定义生命周期：

- 头像：长期保留，旧版本延迟清理
- 聊天图片/视频：按业务保留期管理
- 内容视频：长期或归档保留

## 5. 清理策略

### 5.1 孤儿对象清理

上传成功但未 `complete` 的对象定期清理。

### 5.2 旧头像清理

保留最近 N 个版本，超出后异步清理。

### 5.3 失效派生清理

原对象删除后，派生对象需跟随清理。

## 6. 审计要求

记录：

- 谁初始化上传
- 谁完成上传
- 删除由谁触发
- 资产当前归属对象
