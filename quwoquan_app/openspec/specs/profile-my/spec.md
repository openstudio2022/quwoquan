# profile-my

## 新增需求

### 需求：我的主页与 PROFILE_PAGE_DESIGN v3.0 一致

「我的」频道须与设计文档及 AVATAR_DESIGN_SYSTEM 一致：封面、**圆形头像**（avatar-user-xl，个人头像语义；可点击呼起分身选择抽屉）、名称与简介、主 Tab 为**创作、互动、生活**。创作子分类为**全部、图片、视频、文章**；生活子分类为**足迹、书影音、味蕾、爱物**。小趣入口以常驻圆形图标形式存在于个人资料编辑按钮旁，可管理小趣性格偏好、查看 AI 个人生活报告。

#### 场景：主页内容可见

- **当** 用户处于「我的」频道
- **则** 展示封面、头像（含分身切换）、主 Tab 创作/互动/生活；创作子分类全部/图片/视频/文章；生活子分类足迹/书影音/味蕾/爱物

#### 场景：分身与小趣

- **当** 用户点击头像区域
- **则** 可呼起分身选择抽屉；小趣入口可见，可进入小趣管理

### 需求：我的主页使用语义 token

我的主页上所有 UI 须使用 AppColors、AppSpacing、UITextConstants。禁止使用硬编码值。

#### 场景：我的主页的 token

- **当** 我的主页被构建时
- **则** 所有样式与文案使用设计系统常量
