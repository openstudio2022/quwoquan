# profile-xiaoqu

## 新增需求

### 需求：小趣与设计文档 InformationArchitecture v3.1 一致

小趣须与设计文档及 AVATAR_DESIGN_SYSTEM 一致：图标本意为**星火 (Spark)**，象征灵感的瞬间闪现与兴趣的高光点亮。Slogan：「让兴趣闪亮 (Make Interests Shine)」。拟人化名称为**智多星**。小趣作为 AI 助手，**头像与用户头像统一使用圆形**，尺寸 1:1。交互形态：悬浮球、工具栏中心入口、半屏对话面板（55–60vh）。在趣聊、我的、内容详情页提供入口。

#### 场景：小趣入口可见

- **当** 用户处于趣聊、我的或内容详情页
- **则** 小趣相关入口（悬浮球、工具栏中心、管理入口）可见

#### 场景：小趣半屏面板

- **当** 用户点击小趣入口
- **则** 唤起半屏对话面板，高度 55–60vh；支持向下滑动、背景点击、顶部关闭杆关闭

### 需求：小趣使用语义 token

小趣相关页面与组件所有 UI 须使用 AppColors、AppSpacing、UITextConstants。禁止硬编码。

#### 场景：小趣的 token

- **当** 小趣相关 UI 被构建时
- **则** 所有样式与文案使用设计系统常量
