# comments

## 新增需求

### 需求：从帖子可打开评论视图

应用须支持从帖子（卡片或查看器）打开评论视图。评论视图须展示评论列表与发表评论的输入框。布局须与 Figma 一致。设计文档 v3 已弃用好物标签；评论支持 @ 提及等基础功能。

#### 场景：评论打开

- **当** 用户在帖子上点击评论操作
- **则** 打开评论视图，展示评论列表与输入框

#### 场景：发表评论

- **当** 用户在评论视图中输入文案并提交
- **则** 新增一条评论，列表随之更新

### 需求：评论视图可正确关闭

用户须能关闭评论视图（关闭按钮或滑动）。关闭后用户回到前一上下文。

#### 场景：关闭评论

- **当** 用户关闭评论视图
- **则** 视图按既定动效关闭，前一屏重新可见

### 需求：评论使用语义 token

评论视图中所有 UI 须使用 AppColors、AppSpacing、UITextConstants。禁止使用硬编码值。

#### 场景：评论中的 token

- **当** 评论视图被构建时
- **则** 所有样式与文案使用设计系统常量
