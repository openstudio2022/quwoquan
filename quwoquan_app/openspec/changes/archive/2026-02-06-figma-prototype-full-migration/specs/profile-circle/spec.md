# profile-circle

## 新增需求

### 需求：圈子主页与 CIRCLE_DESIGN v2.0 一致

圈子主页须与设计文档及 AVATAR_DESIGN_SYSTEM 一致：**圆角正方形头像**（avatar-circle-xl，圈子头像语义）、子频道含**创作、讨论、生活、聊天**。主 3-Tab 为**创作、互动、生活**，与个人主页统一。创作 Tab 子分类为全部/图片/视频/文章；生活 Tab 展示圈子成员共同标记的兴趣点与爱物。统计与加入/关注等操作与设计及原型一致。

#### 场景：圈子布局

- **当** 圈子主页被展示时
- **则** 主 Tab 为创作/互动/生活；创作子分类为全部/图片/视频/文章；生活展示共同兴趣点与爱物

#### 场景：主操作为关注与加入

- **当** 用户为访客时
- **则** 显示关注圈子/已关注、加入圈子/已加入等；管理员可显示编辑圈子、管理中心

#### 场景：返回关闭叠加层

- **当** 用户点击返回
- **则** 圈子主页关闭，并恢复前一上下文

### 需求：吸顶 Tab 与内容区

滚动时主 Tab 栏须吸顶。创作 Tab 下展示圈子内作品；互动 Tab 下展示赞/评论等；生活 Tab 展示共同兴趣与爱物。

#### 场景：Tab 吸顶与内容

- **当** 用户滚动圈子主页
- **则** Tab 栏吸顶；内容区随 Tab 切换

### 需求：圈子主页使用语义 token

圈子主页上所有 UI 须使用 AppColors、AppSpacing、UITextConstants。禁止使用硬编码值。

#### 场景：圈子主页的 token

- **当** 圈子主页被构建时
- **则** 所有样式与文案使用设计系统常量
