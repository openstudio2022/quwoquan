## 1. 外壳与三段式布局

- [x] 1.1 实现图片编辑器页面外壳：顶栏固定、中部占满剩余高度、底栏固定，三区边界清晰无重叠
- [x] 1.2 顶栏左侧返回、中间图片序号占位、右侧完成按钮（使用 AppColors.primaryColor）
- [x] 1.3 底栏编辑功能栏骨架：旋转、裁剪、滤镜、专业修图、相框、文字、涂鸦、马赛克入口，支持横向滚动
- [x] 1.4 所有 UI 使用 AppColors、AppSpacing、UITextConstants，无硬编码

## 2. 顶栏与编辑历史（Snapseed 式）

- [x] 2.1 实现编辑步骤栈：每步记录工具类型与参数快照，支持追加、删除某步、重算其后步骤
- [x] 2.2 顶栏或底栏提供历史入口，打开步骤列表可查看、删除某步、或重做某步（进入该步编辑并保存后更新）
- [x] 2.3 完成按钮点击前可访问历史；点击完成后提交结果并关闭编辑器（与 create-flow 约定一致）

## 3. 中部图片与多图滑动

- [x] 3.1 中部展示当前图片，占满可用垂直空间；单图时顶栏序号为 1/1
- [x] 3.2 多图时中部使用 PageView（或等价）左右滑动切换图片，顶栏序号随当前页更新为「当前/总数」

## 4. 操作面板通用结构

- [x] 4.1 底栏选中某一功能（旋转、裁剪、滤镜、相框、文字、涂鸦、马赛克之一）时，在底栏上方展示操作面板
- [x] 4.2 面板结构：顶部为分类/选项、中间为模板或内容、底部为取消（叉）与保存（勾），底部中间为可选比例/强度等控件
- [x] 4.3 取消时放弃当前工具子操作并关闭面板；保存时将当前操作作为一步写入历史并关闭面板

## 5. 滤镜与裁剪面板

- [x] 5.1 选中滤镜：面板顶部滤镜分类、中间滤镜模板列表与预览、底部取消/保存及强度滑块
- [x] 5.2 选中裁剪：图片上可调裁剪框与比例选项（自由、原图、正方形、3:2 等），面板底部取消/保存

## 6. 旋转工具与旋转仪表、网格

- [x] 6.1 选中旋转：图片下方展示旋转仪表（约 ±45° 或更大范围），底部保留取消/保存及向左 90°、向右 90°、水平翻转、垂直翻转等快捷操作
- [x] 6.2 调节旋转时在图片上叠加对齐网格（如等分网格），便于对齐主体与角度

## 7. 专业修图与曲线蒙皮

- [x] 7.1 底栏「专业修图」入口进入后展示子工具列表：曲线、白平衡、局部、修复、美丽光晕、色调对比度、HSL 调色
- [x] 7.2 曲线子工具：曲线调节控件（曲线图、通道选择、控制点）以蒙皮形式展示在图片下方区域，底部取消/保存；保存时该步写入历史
- [x] 7.3 其他专业修图子工具（白平衡、局部、修复、美丽光晕、色调对比度、HSL）具备基础入口与取消/保存，可先占位或对接既有实现

## 8. 相框、文字、涂鸦、马赛克

- [x] 8.1 相框、文字、涂鸦、马赛克各工具在选中时按统一面板结构展示对应分类/模板与取消/保存，具体交互可先实现基础版本

## 9. 与创作流集成与联调

- [x] 9.1 创作流图片 Tab 使用本编辑器作为唯一图片编辑入口（传入图片列表、接收完成/取消回调与导出结果）
- [x] 9.2 完成时导出编辑结果（含历史步骤应用）并返回创作流；取消时无保存并关闭
